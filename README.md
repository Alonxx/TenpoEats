### Installation 💻

- run `npm install` or `yarn install`

### If you are on Mac 🍎
 
 - run `pod install` in ios project folder
 
 ### Run on Device ⚒️

- run `npm run android` - `yarn run android` or  -`npm run ios` - `yarn run ios` to run on device or emulator

### Technologies used 🔥

- React Native
- Styled-components
